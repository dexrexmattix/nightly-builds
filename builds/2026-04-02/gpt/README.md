This is a simple web page that helps Codi input a list of daily tasks and uses OpenAI's GPT-4 API to automatically generate a prioritized to-do list. To use it, open the index.html file in a modern web browser, enter your tasks one per line, and click "Generate Prioritized List". You will be prompted to enter your OpenAI API key (you must have one). The page will then display your tasks ranked by priority for the day, saving you time and mental effort organizing your schedule.

To run:
1. Open index.html in a browser.
2. Enter your tasks in the text area.
3. Click the button and enter your OpenAI API key when prompted.
4. View your prioritized to-do list below.

No server or installation needed. Requires internet access for the AI API call.
